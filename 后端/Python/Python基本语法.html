# 注释

# 单行注释

使用“#”开头，为了保证代码的可读性，推荐#号与注释内容之间有一个空格 可位于单独的一行，也可以位于代码的后面，此时推荐代码与注释的#号之间有两个空格

# 多行注释

使用一对连续的三个引号，单双引号均可，比如：

'''
这是多行注释
'''

"""
这是多行注释
"""

# Python代码规范

  • Python 官方提供有一系列 PEP（Python Enhancement Proposals） 文档

  • 其中第 8 篇文档专门针对 Python 的代码格式 给出了建议，也就是俗称的 PEP 8

  • 文档地址：https://www.python.org/dev/peps/pep-0008/

  • 谷歌有对应的中文文档：http://zh-google-styleguide.readthedocs.io/en/latest/google-python-styleguide/python_style_rules/

# 运算符

# 算术运算符

完成基本的算术运算使用的符号，用来处理四则运算

运算符 描述 实例
+ 加 10 + 20 = 30
- 减 10 - 20 = -10
* 乘 10 * 20 = 200
/ 除（商总是浮点数） 10 / 20 = 0.5、2 / 1 = 2.0
// 取整除 返回除法的整数部分（商） 9 // 2 输出结果 4
% 取余数 返回除法的余数 9 % 2 = 1
** 幂 又称次方、乘方，2 ** 3 = 8

在 Python 中 * 运算符还可以用于字符串，计算结果就是字符串重复指定次数的结果

In [1]: "-" * 50
Out[1]: '----------------------------------------'

# 比较（关系）运算符

运算符 描述 实例
< 小于
<= 小于等于
> 大于
>= 大于等于
== 等于
!= 不等于
is 判断两个标识符是否引用自一个对象 x is y, 如果id(x) == id(y)，即x也y的指向同一个内存地址，则结果为1，否则结果为0
is not 判断两个标识符是否引用自不同对象 x is not y, 如果id(x) != id(y)，即x和y指向不同的内存地址，则结果为1，否则结果为0

# 赋值运算符

运算符 描述 实例
= 简单赋值
+= 加法赋值 a += b 相当于 a = a + b
-= 减法赋值 a -= b 相当于 a = a - b
*= 乘法赋值 a *= b 相当于 a = a * b
/= 除法赋值 a /= b 相当于 a = a / b
//= 取整除赋值 a //= b 相当于 a = a // b
%= 取模赋值 a %= b 相当于 a = a % b
**= 幂赋值 a **= b 相当于 a = a ** b

# 逻辑运算符

运算符 描述 实例
and 布尔与 a and b，两者都为True，结果才返回True，否则返回False
or 布尔或 a or b，至少有一个为True，结果就返回True，否则返回False
not 布尔非 a -= b 相当于 a = a - b

# 位运算符

运算符 描述 实例
& 按位与 两个值的两个相应位都为1，则该位的结果为1；否则为0
| 按位或 两个值的两个相应位有一个为1，则该位的结果为1；否则为0
~ 按位取反 对数据的每个二进制位取反，即把 1 变为 0 ，把 0 变为 1
^ 按位异或 当两个对应的二进制位不相等时，结果为1，相等结果为0
>> 按位右移 运算数的各个二进制位全部右移若干位
<< 按位左移 运算数的各个二进制位全部左移若干位，高位丢弃，地位不补 0

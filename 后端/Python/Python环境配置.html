# Python环境配置

本人使用的macOS上自带了Python2.x版本，自带的Python2的安装路径：

192:~ dengyouxu$ which python
/usr/bin/python

如果想装Python3.x的话，有以下几种方式：

  1. Homebrew安装
  2. Python官网下载pkg进行安装

安装过程略，安装后Python3后，查看安装路径：

192:~ dengyouxu$ which python3
/Library/Frameworks/Python.framework/Versions/3.7/bin/python3

# Python的包管理工具pip

pip是常用的python包管理工具，类似于java的maven。用python的同学，都离不开pip。

# Python的增强型解释器ipython

IPython 中 的 “I” 代表 交互 interactive

# 特点

  • IPython 是一个 python 的 交互式 shell，比默认的 python shell 好用得多
    • 支持自动补全
    • 自动缩进
    • 支持 bash shell 命令
    • 内置了许多很有用的功能和函数
  • IPython 是基于 BSD 开源的

# 版本

  • Python 2.x 使用的解释器是 ipython
  • Python 3.x 使用的解释器是 ipython3

# 退出解释器

  • 直接输入 exit

    In [1]: exit

  • 使用热键退出 在 IPython 解释器中，按热键 ctrl + d，IPython 会询问是否退出解释器

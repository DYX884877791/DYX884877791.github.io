# String 字符串

# 字符串如何定义

  1. 在Python中，可以用单引号或者双引号' '、""来定义一个字符串类型的变量，除此之外，还可以使用一对三个连续的引号来定义一串占用多行的字符串，比如'''...'''或者"""..."""；

  2. 在Python3.x版本中，字符串是以Unicode编码的，也就是说，Python3.x中的字符串默认支持多语言，而在Python2.x中，字符串类型是以ASCII编码的，默认只支持英文，当需要定义中文字符串变量时，需要在文件头部指明文件编码格式为UTF-8；

  3. 如果字符串内容本身包含了单 引号或双引号，此时就需要进行特殊处理 。此时可以使用不同的引号将字符串括起来处理。 也可以对引号进行转义（使用反斜线"\"将字符串中的特殊字 符进行转义）；

str1 = "I'm a Python Coder"
print(str1)
str2 = 'I\'m a Python Coder'
print(str2)

# 长字符串

  1. 当程序中有大段文本内容或者多行文本内容要定义成字符串时，可以使用连续三个引号'''...'''或者"""..."""定义长字符串，因为这种形式非常强大，可以让字符串中包含任何内容，既可包含单引号，也可包含双引号 ，优先推荐使用长字符串形式，定义的长字符串打印输出时按原样输出；
str3 = '''The quick brown fox
jumps over the lazy dog'''
print(str3)

The quick brown fox
jumps over the lazy dog
  1. Python 还允许使用转义字符"\"对换行符进行转义，转义之后的换行符不会“中断” 字符串 ；
str4 = 'The quick brown fox \
jumps over the lazy dog'
print(str4)

The quick brown fox jumps over the lazy dog
  1. 如果在长字符串中每一行字符串的行尾使用转义字符"\"时，相当于对行尾的换行符也进行了转义，这样会在同一行中输出；
str5 = '''The quick brown fox \
jumps over the lazy dog'''
print(str5)

The quick brown fox jumps over the lazy dog

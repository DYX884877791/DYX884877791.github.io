# Centos7安装MySQL5.7

  2. 卸载Centos7自带的mariadb

    [root@hserver1 opt]# rpm -qa | grep -i mariadb
mariadb-libs-5.5.52-1.el7.x86_64
[root@hserver1 opt]# rpm -e --nodeps mariadb-libs-5.5.52-1.el7.x86_64

  3. 安装MySQL

    [root@hserver1 opt]# rpm -ivh mysql-community-common-5.7.21-1.el7.x86_64.rpm mysql-community-libs-5.7.21-1.el7.x86_64.rpm mysql-community-client-5.7.21-1.el7.x86_64.rpm mysql-community-server-5.7.21-1.el7.x86_64.rpm

    安装之后，MySQL各目录为：

    1. 数据库目录：/var/lib/mysql/
    2. 命令配置：/usr/share/mysql  (mysql.server命令及配置文件)
    3. 相关命令：/usr/bin   (mysqladmin mysqldump等命令)
    4. 启动脚本：/etc/rc.d/init.d/   (启动脚本文件mysql的目录)
    5. 系统配置：/etc/my.conf

  4. 为了保证数据库目录为与文件的所有者为 mysql 登陆用户，如果你是以 root 身份运行 mysql 服务，需要执行下面的命令初始化

    [root@hserver1 opt]# mysqld --initialize --user=mysql

  5. 启动mysql服务，使用命令

    [root@hserver1 opt]# systemctl start mysqld.service

    或者

    启动：

    使用 service 启动：service mysqld start
使用 mysqld 脚本启动：/etc/init.d/mysqld start
使用 safe_mysqld 启动：safe_mysqld&

    停止：

    使用 service 启动：service mysqld stop
使用 mysqld 脚本启动：/etc/init.d/mysqld stop
mysqladmin shutdown

    重启：

    使用 service 启动：service mysqld restart
使用 mysqld 脚本启动：/etc/init.d/mysqld restart

  6. 初始化会为root用户生成一个随机密码，位于/var/log/mysqld.log日志文件中

    [root@hserver1 opt]# cat /var/log/mysqld.log | grep password

  7. 使用上一步的密码登录到MySQL的命令客户端，并输入密码

    [root@hserver1 opt]# mysql -u root -p
Enter password:

  8. 修改密码，假如修改为123456

    mysql> set password for root@localhost = password('123456');

  9. 授权远程连接

    mysql> GRANT ALL PRIVILEGES ON . TO 'root'@'%' IDENTIFIED BY ‘123456’ WITH GRANT OPTION;
Query OK, 0 rows affected, 1 warning (0.01 sec)
mysql> FLUSH PRIVILEGES;
Query OK, 0 rows affected (0.01 sec)

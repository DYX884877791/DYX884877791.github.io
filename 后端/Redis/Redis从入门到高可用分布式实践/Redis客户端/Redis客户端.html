# Redis客户端

# Java客户端：Jedis

  1. 获取Jedis，通过maven引入

    <dependency>
   <groupId>redis.clients</groupId>
   <artifactId>jedis</artifactId>
   <version>2.9.0</version>
</dependency>

  2. jedis的使用

    1. 生成一个Jedis对象，该对象负责和指定的Redis节点进行通信
    2. Java直接调用进行Redis的API进行操作
      public static void main(String[] args) {
      // 指定Redis节点的ip和端口
      Jedis jedis = new Jedis("192.168.13.231",6381);

      // jedis执行set操作，并返回结果
      String setResult = jedis.set("hello", "world");
      System.out.println(setResult);

      // jedis执行get操作，并返回结果
      String getResult = jedis.get("hello");
      System.out.println(getResult);
      
      Long hsetResult = jedis.hset("hashkey", "hashfield", "hashvalue");
      System.out.println(hsetResult);
      String hgetResult = jedis.hget("hashkey", "hashfield");
      System.out.println(hgetResult);
      
      Long lpushResult = jedis.lpush("listkey", new String[]{"listvalue1", "listvalue2"});
      System.out.println(lpushResult);
      List<String> lrangeResult = jedis.lrange("listkey", 0, -1);
      System.out.println(lrangeResult);

      Long saddResult = jedis.sadd("setkey", new String[]{"setvalue1", "setvalue2"});
      System.out.println(saddResult);
      Set<String> smembersResult = jedis.smembers("setkey");
      System.out.println(smembersResult);

      Long zaddResult = jedis.zadd("zsetkey", 23.5, "zsetelement");
      System.out.println(zaddResult);
      Set<String> zrangeResult = jedis.zrange("zsetkey", 0, -1);
      System.out.println(zrangeResult);
}

# jedis连接池的使用

当使用jedis直连Redis时，Jedis是基于socket连接，每次生成一个Jedis对象时，需要创建连接，使用完毕需要将它关闭，比较耗费资源，所以可以使用Jedis连接池 Jedis连接池JedisPool
